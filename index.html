How this works — 3 steps
1. Import Sheet — drop your drawings (SVG, photo, blueprint). Drop several; each becomes a page.
2. Reference & Trace — draw one loop around the shape in each view: Side (from the side), Top (from above), Front (head-on).
3. 3D Preview — your object appears. Drag to spin it.
Your drawing always wins. The sliders below are just a rough lump to start from before you trace.
Rough starting proportions:
The finished measurements. After you trace, these show what your drawing measured — move them to resize and your tracing is left alone.
2Side profile · how tall▶
From: the sliders — trace a side view to use your drawing instead.
3Width · how wide▶
From: the sliders — trace a top view to use your drawing instead.
4Shape style · smooth or exact▶
Smooth — a flowing, rounded body. Best for cars, boats, anything streamlined.
Follow my drawing — carves out exactly what your outlines say, corners and all. Use this when Smooth will not hold the shape.
⚙Advanced · detail & thickness▶
Only touch these if you want to. Higher detail = smoother but slower.
5Cross-section · the shape end-on▶
A cross-section is pinned at each cut; the model morphs between them. Front & rear views become cuts automatically. Drag the dots to reshape the selected cut — this edits the 3D model live.
From: the sliders — trace a front view for the real end-on shape.
No section cuts yet — trace a front view, import a sheet, or add one.
6Save & export▶
Keeps it on this device and in the shared cloud, and it appears in the Workshop. Saving again with the same name updates it instead of making a copy.
Optional extras that need the backend service running. Build on server makes a STEP file for proper CAD tools. Without a backend these just download the file locally instead — nothing breaks.
☁Cloud saves▶
Save projects (full trace + model) to a Supabase (Postgres) database so they persist and sync — no backend server to run. Best hard-wired via GitHub repo secrets (see SUPABASE.md) so no keys live in the app. To try locally, paste your URL + anon key below. ▸ show setup SQL
